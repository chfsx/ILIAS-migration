transform

private void transform( object domNode $node, [ integer $left2, integer $lvl ] )
  traverse the domDocument and build a tree which contains
additional information about domDocument's structure:$arr[id] = array( content (str) = node_value (only text nodes have a value)name (str) = tagname or entitynametype (int) = element_type_iddepth (int) = depth of node in treeparent (int) = id of parent nodefirst (int) = id of first child nodeprev (int) = id of previous sibling nodenext (int) = id of next sibling nodeleft (int) = left value (for traversing tree in relational DB)right (int) = right value (for traversing tree in relational DB))The key is the internal id of the domDocument. Also the ids of other nodes are internal references.The array is written to $this->tree. Use $this->buildTree() to return the variable.attributes are NOT converted yet!

Parameter
object domNode $node
integer $left2 = >>-1<<
value (optional; only needed for recursion))
integer $lvl = >>0<<
of node in tree (optional, default is 0)
Returns void
