ILIAS allows the use of multiple implementations of the SCORM RTE.

The RTE used here is fully SCORM-1.2 conformant. It is based on the PfPLMS SCORM-1.2 API-Adapter Core,
Copyright (c) Matthai Kurian, ETHZ (Swiss Federal Institute of Technology, Zürich), Switzerland

The PfPLMS SCORM-1.2 API-Adapter Core is "Free Software", published under the terms of the GNU General Public License (GPL) on the International Relations and Security Network (ISN) web site.

Together with ILIAS' SCORM-1.2 import capability, this RTE makes ILIAS a fully SCORM-1.2 conformant Learning Management System.

See the source code of the API-Adapter Java Applet IliasApiAdapterApplet.java.
