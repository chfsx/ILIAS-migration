Changes to the DTD:
-------------------

Paragraph -> new attribute: Characteristic (Example | Citation | Mnemonic) #IMPLIED
Contribute -> attribute Role: double entry for validator
Title -> (%titletext;)* --> (#PCDATA)
Description -> (%simpletext;)* --> (#PCDATA)
Taxon -> (Entry) --> (#PCDATA), new attribute: Language (%langcode;) #REQUIRED
Entry -> removed (not needed at all)
Educational -> Language+ removed (obsolete)
Definition -> (Paragraph+) --> (Paragraph | LearningObject)+
Answer -> new attribute: Status (Right | Wrong) #IMPLIED
Paragraph -> extended attribute: Characteristic (Headline | Example | Citation | Mnemonic) #IMPLIED
Content -> (Paragraph | LearningObject) --> (Paragraph | LearningObject)
Text -> removed (not needed at all)
Headline -> removed (not needed at all)
